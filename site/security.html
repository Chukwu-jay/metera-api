Security

Security Posture

Metera is built for careful beta rollout. The default posture favors conservative reuse, explicit user actions, tenant/workspace boundaries, and auditable policy decisions.

Current Controls

  • semantic hardening presets with Conservative as the recommended beta default;
  • namespace, tenant, and workspace scoping for durable workflow memory;
  • DLP and secret scrubbing before semantic indexing;
  • request ledger, policy, and savings visibility;
  • browser extension actions that require explicit user clicks;
  • local preview before browser captures are saved to Metera.

Security Reports

To report a suspected vulnerability, unsafe cache reuse, or tenant isolation issue, email support@getmetera.com.
